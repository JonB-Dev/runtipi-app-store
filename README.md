# JonB Runtipi App Store

A personal [runtipi](https://runtipi.io) app store.

Add it in runtipi: Settings → App Stores → Add App Store, using this repo's URL.

## Apps
- **Budget Manager** (`bill-manager`) — self-hosted bill & budget manager with passkey sign-in. Image: `ghcr.io/jonb-dev/bill-mgr:latest`.
